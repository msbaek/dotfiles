## 정리 가이드

### Homebrew로 이전 가능한 앱

다음 앱들은 현재 직접 다운로드되어 있지만 Homebrew Cask로 관리 가능:

```bash
# 예시 - Brewfile에 추가 가능한 앱들
brew install --cask alfred
brew install --cask appcleaner
brew install --cask daisydisk
brew install --cask obsidian
brew install --cask notion
brew install --cask slack
```

### 자동 업데이트 관리

- **Homebrew Cask**: `brew upgrade --cask` 로 일괄 업데이트
- **Mac App Store**: `mas upgrade` (mas-cli) 로 자동 업데이트
- **JetBrains Toolbox**: 자체 업데이트 관리
- **회사 배포**: IT 부서 관리 (자동 업데이트 비활성화)

### 디스크 공간 정리 우선순위

1. Xcode (수십 GB): `xcrun simctl delete unavailable` - 미사용 시뮬레이터 삭제
2. OrbStack/Docker 이미지: `orbstack clean` - 미사용 컨테이너/이미지
3. ScreenFlow 프로젝트 파일: 편집 완료 후 내보내기 후 삭제
4. 전자책 뷰어 캐시: AladinEbook, YES24, KBSKong 캐시 정리
